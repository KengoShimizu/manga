# 漫画ビューワ
某IT会社の2日間ハッカソン形式のインターンシップで，apiを使って漫画情報（テキスト・画像）を取得する漫画ビューワを作成した．
apiはインターンシップ専用に用意されたものなので，将来的に削除されるため　以下に画像とgifで記録しておく．
## 使用技術
